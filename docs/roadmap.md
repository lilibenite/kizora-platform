# KIZORA Development Roadmap

KIZORA – Institute of Space & Technology  
Founder: TWAGIRAYO Lili Benite

## Goal
Build a working prototype of the KIZORA learning platform within six months and get real students using it.

## Month 1 – Planning and Foundations
- Define KIZORA vision and architecture
- Set up GitHub project
- Design platform structure

## Month 2 – Platform Setup
- Build basic website structure
- Create homepage and navigation
- Prepare course system

## Month 3 – User System
- Add login and signup
- Create student dashboard
- Store user data

## Month 4 – Learning Features
- Add lessons
- Add quizzes
- Track student progress

## Month 5 – Testing with Users
- Invite students to try the platform
- Collect feedback
- Improve usability

## Month 6 – Final Improvements
- Improve interface
- Fix bugs
- Prepare documentation and impact report
