# KIZORA MVP (Minimum Viable Product)

KIZORA – Institute of Space & Technology  
Founder: TWAGIRAYO Lili Benite

## Objective

Build the first working version of the KIZORA learning platform and test it with real students.

## Core Features

### 1. User Accounts
Students can create accounts and log in to the platform.

### 2. Course Page
Students can view available courses.

### 3. Lessons
Each course contains lessons explaining scientific concepts.

### 4. Quiz System
Students answer questions to test their understanding.

### 5. Progress Tracking
The system tracks student progress through lessons.

## First Course

Introduction to Space Science

Topics may include:
- What is space
- Satellites and orbits
- Basics of rockets
- Astronomy fundamentals

## Success Criteria

The MVP will be considered successful if:
- the platform works without major errors
- at least 20–50 students use it
- feedback is collected from users
