# KIZORA Platform Architecture

KIZORA – Institute of Space & Technology  
Founder: TWAGIRAYO Lili Benite

## Overview

KIZORA will be a web-based learning platform focused on science, technology, and space education.

The system will be built using a three-layer architecture.

## Frontend

The frontend is the user interface that students interact with.

Technologies:
- HTML
- CSS
- JavaScript

Responsibilities:
- login page
- course interface
- lessons and quizzes
- student dashboard

## Backend

The backend handles application logic and data processing.

Technology:
- Python (FastAPI or Django)

Responsibilities:
- user authentication
- storing course content
- quiz processing
- tracking student progress

## Database

The database stores all platform information.

Technology:
- PostgreSQL

Data stored:
- users
- courses
- lessons
- quiz results
- progress tracking

## Future Expansion

Future versions of KIZORA may include:

- AI learning assistant
- space science simulations
- satellite data exploration tools
